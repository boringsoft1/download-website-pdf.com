# Privacy Policy for "Download Website PDF"

Effective date: 13.06.2026

## 1. Overview

"Download Website PDF" is a browser extension that allows users to generate and download a PDF version of the currently open webpage.  
We respect your privacy. This extension does not collect personally identifiable information and does not track users across websites.

## 2. Information We Collect

We collect limited, anonymous technical data to improve the functionality and reliability of the extension.
This may include:
- Anonymous installation identifier (random UUID)
- Anonymous session identifier
- Extension version
- Browser language (navigator.language)
- Selected extension language
- Current page domain (e.g., example.com)
- Truncated page path without query parameters (e.g., /article/123). Query parameters (such as ?id=..., ?email=..., or tracking tokens) are explicitly removed before any data is sent.
- Viewport size (width and height)
- Interaction events (e.g., button clicks, onboarding steps, download actions)

We intentionally exclude URL query parameters to prevent accidental collection of sensitive or personal information.  
The extension does not track user activity across different websites.  
This data cannot be used to identify you personally.  

## 3. How We Use Information

We use collected data solely to:
- Improve extension performance and stability
- Understand feature usage
- Detect and fix errors
- Optimize user experience
We do not use this data for advertising or profiling.  
Data is collected only when the user interacts with the extension (e.g., clicking the download button or using the context menu).

## 4. Data Storage

- Data is transmitted securely to our servers:
  - https://ext.api.boringsoft.io
  - https://analytics.download-website-pdf.com
- All data is transmitted over secure HTTPS connections.
- Data is stored in aggregated and anonymized form
- No personal accounts are created
- No sensitive browsing data (such as page content, form inputs, passwords, or payment information) is collected

## 5. No Personal Data Collection

We explicitly do not collect:
- Names
- Email addresses
- IP addresses are not stored, tracked, or associated with user identities.
- Exact location
- Browsing history across sites
- Page content
- Form data

The content of web pages is processed locally in the user's browser to generate the PDF file and is never transmitted to any server.

## 6. Data Sharing

We do **not sell**, **rent**, **or share** your data with third parties.  
Data is used only by the developer to maintain and improve the extension.

## 7. User Control

You can stop all data collection by uninstalling the extension at any time.

## 8. Security

We take reasonable measures to protect collected data from unauthorized access or misuse.  
We do not use remote code execution. 

## 9. Changes to This Policy

This Privacy Policy may be updated from time to time. Updates will be reflected on this page with a new effective date.
